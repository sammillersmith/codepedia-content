Defines a row in a table.

## Syntax

```html
<tr>
  <!-- Takes in row contents -->
</tr>
```

## Example 1

Creating three rows for a table.

```html
<table>
  <!-- Row 1 -->
  <tr>
    <th>Company</th>
    <th>Phone</th>
  </tr>

  <!-- Row 2 -->
  <tr>
    <td>Apple</td>
    <td>iPhone</td>
  </tr>

  <!-- Row 3 -->
  <tr>
    <td>Samsung</td>
    <td>Galaxy</td>
  </tr>
</table>
```

| Company | Phone  |
| ------- | ------ |
| Apple   | iPhone |
| Samsung | Galaxy |
