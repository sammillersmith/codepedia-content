Defines the width of the bottom padding for an element.

## Syntax

```css
padding-bottom: <value>;
```

where `<value>` can be one of the following:

- Length: `20px`
- Percentage: `5%`

## Example 1

Set the bottom padding of the `h1` element to be `20px`:

```css
h1 {
  padding-bottom: 20px;
}
```

## Example 2

Set the bottom padding of the `h1` element to be `5%` relative to the width of the containing block:

```css
h1 {
  padding-bottom: 5%;
}
```
