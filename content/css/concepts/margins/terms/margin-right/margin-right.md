Defines the right margin area for an element.

## Syntax

```css
margin-right: <value>;
```

where `<value>` can be one of the following:

- Length: `20px`
- Percentage: `5%`

**Note:** values provided may be negative, placing the element closer.

## Example 1

Set the right margin of the `span` element to be `20px`:

```css
span {
  margin-right: 20px;
}
```

## Example 2

Set the right margin of the `span` element to be `5%`:

```css
span {
  margin-right: 5%;
}
```
