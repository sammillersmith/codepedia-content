The **`acos()`** method of the Lua math library returns the arc cosine of the argument, in radians.

## Syntax

```pseudo
math.acos(x)
```

Where _-1 <= x <= 1_.

## Example

The code below will print the arc cosine of `0.5` to the console.

```lua
print(math.acos(0.5))
```

The output will be:

```shell
1.0471975511966
```
