The Python `.clear()` method removes all items from a list.

## Syntax

```py
list.clear()
```

The `.clear()` method has no parameters.

## Example

To remove all items from the `grocery` list:

```python
grocery = ['🍉', '🍪', '🥬', '🥕']

grocery.clear()

print(grocery)
# Output: []
```

## Codebyte Example

To remove all items from the `orders` list:

```codebyte/python
orders = ['daisies', 'periwinkle']

orders.clear()

print(orders)
```
