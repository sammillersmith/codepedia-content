The `.keys()` method of a Python dictionary returns list of its keys. It takes no arguments.

## Syntax

```pseudo
dictionary.keys()
```

## Codebyte Example

The following example creates a dictionary then prints all of its keys:

```codebyte/python
d = {1:'one', 2:'two', 3:'three'}
d_keys = d.keys()
print(d_keys)
```
