Converts an integer into its binary equivalent string.

## Syntax

```py
bin(integer)
```

## Example 1

Use `bin()` function to return the binary equivalent of the integer `1`:

```python
print(bin(1))

# Output: 0b1
```

## Example 2

Use `bin()` to return the binary equivalent of the integer `100`:

```codebyte/python
print(bin(100))
```
