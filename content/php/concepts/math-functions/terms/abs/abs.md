The **`abs()`** function returns the absolute value of the argument.

## Syntax

```pseudo
$result = abs(n);
```

Where `n` is the number to find the absolute value of.

## Codebyte Example

Using `abs()` to return the absolute value of `-6.5`:

```codebyte/php
<?php
  echo(abs(-6.5));
?>
```
